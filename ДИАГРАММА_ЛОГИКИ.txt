ЛЕГЕНДА:

✅ УСПЕХ        = Получен текстовый ответ → вернуть
❌ TIMEOUT      = Прошло 15 сек → попробовать снова/переключиться
⚠️ EXCEPTION    = Ошибка API (Connection error и т.д.) → попробовать снова
🔄 RETRY        = Повтор той же системы
🔀 FALLBACK     = Переключение на другую систему
🌐 LANGUAGE     = Определение казахского (әіңғүұқө) vs русского

═══════════════════════════════════════════════════════════════════

ВРЕМЕННАЯ ШКАЛА:

Сценарий 1: DeepSeek успешен
  DeepSeek попытка 1 → [1-3сек] → ✅ УСПЕХ → [0.5сек] → Клиенту
  ИТОГО: 2-3.5 сек ✅ Быстро

Сценарий 2: DeepSeek timeout → OpenAI спасает
  DeepSeek попытка 1 → [15сек timeout]
  DeepSeek попытка 2 → [15сек timeout]
  Переключение → [0.5сек]
  OpenAI попытка 1 → [1-3сек] → ✅ УСПЕХ
  ИТОГО: 15+15+0.5+3 = 33.5 сек ⚠️ Долго, но работает

Сценарий 3: Все не работают
  DeepSeek попытка 1 → [15сек]
  DeepSeek попытка 2 → [15сек]
  OpenAI попытка 1 → [15сек]
  OpenAI попытка 2 → [15сек]
  Fallback текст → [0.1сек] → ✅ УСПЕХ
  ИТОГО: 15+15+15+15+0.1 = 60.1 сек ⚠️ Очень редко

═══════════════════════════════════════════════════════════════════

УСЛОВИЯ ПЕРЕКЛЮЧЕНИЯ:

1. DeepSeek → OpenAI
   ├─ Если обе попытки DeepSeek вернули пусто/timeout/error
   ├─ ИЛИ первая попытка успешна → не переключаемся
   └─ ИЛИ вторая попытка успешна → не переключаемся

2. OpenAI → Fallback
   ├─ Если обе попытки OpenAI вернули пусто/timeout/error
   ├─ Язык определяется по последнему сообщению клиента
   │  ├─ Если есть казахские буквы (әіңғүұқө) → казахский фолбэк
   │  └─ Иначе → русский фолбэк
   └─ Фолбэк текст всегда возвращает ответ (никогда не выбросит исключение)

ПРАКТИЧЕСКИЕ ПРИМЕРЫ:

📱 Клиент пишет: "Привет"
   → DeepSeek за 2сек → ✅ "Привет! Меня зовут Луна..."
   ┗ Логи: ✅ DeepSeek успешен (попытка 1)

📱 Клиент пишет: "Какие тарифы?" (во время перегруза DeepSeek)
   → DeepSeek timeout (15сек)
   → DeepSeek timeout (15сек)
   → OpenAI за 2сек → ✅ "Курсы разделены на офлайн и онлайн..."
   ┗ Логи: ⚠️ DeepSeek дважды пустой, переключаемся на OpenAI
          ✅ OpenAI успешен (попытка 1)

📱 Клиент пишет: "Сәлем" (казахский, во время отказа обоих)
   → DeepSeek ошибка ✗
   → DeepSeek ошибка ✗
   → OpenAI ошибка ✗
   → OpenAI ошибка ✗
   → Fallback казахский → ✅ "Кешіріңіз, қате болды..."
   ┗ Логи: ❌ Оба провайдера не смогли ответить, используем fallback

═══════════════════════════════════════════════════════════════════

СТАТУСЫ В КОДЕ:

✅ SUCCESS    = text_parts не пуста, возвращаем результат
❌ EMPTY      = msg.content пуста или нет текстовых блоков
⏱️  TIMEOUT   = asyncio.TimeoutError после 15 сек
⚠️  EXCEPTION = Любая другая ошибка (ConnectionError, APIError и т.д.)

Все три типа (EMPTY, TIMEOUT, EXCEPTION) обрабатываются одинаково:
возвращают None → переход к следующей попытке/провайдеру
