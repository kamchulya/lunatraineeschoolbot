# 🌙 Luna Bot — Champion School

ИИ-ассистент для Instagram Direct канала Champion School (Школа фитнеса).
Построен на Claude Haiku, FastAPI (aiohttp), PostgreSQL, интеграция с Wazzup и EnvyCRM.

---

## Архитектура

```
Клиент → Instagram → Wazzup → EnvyCRM (создаёт лид)
                                    ↓
                          EnvyCRM шлёт хук на /envy_hook
                                    ↓
                              Луна отвечает
                          (через Wazzup API)
```

---

## Стек

- **Python 3.11+** — aiohttp, asyncpg, anthropic, apscheduler
- **Claude Haiku 4.5** — основная модель (temperature=0.2)
- **PostgreSQL** — хранение диалогов и состояний
- **Wazzup** — Instagram Direct интеграция
- **EnvyCRM** — CRM, лиды, менеджеры
- **Google Sheets API** — живая база знаний (обновляется каждые 30 мин)
- **APScheduler** — напоминания после демо, фоновые задачи

## States (состояния диалога)

| State | Описание |
|-------|----------|
| `new` | Первое сообщение |
| `active` | Активный диалог с ботом |
| `demo_sent` | Отправлен демо-доступ, ждём реакции |
| `manager` | Менеджер взял чат, бот молчит |
| `done` | Сделка закрыта |

## Напоминания после демо

После отправки демо-доступа бот автоматически планирует 3 напоминания:
- **+1 час** — "Как впечатления от платформы?"
- **+3 часа** — "Вы можете начать обучение уже сейчас"
- **+10 часов** — "Удалось посмотреть материалы?"

Если клиент написал сам — все напоминания отменяются.

---

## Автоответ при молчании менеджера

Если менеджер взял чат (STATE_MANAGER) но не ответил клиенту в течение **30 минут** — бот автоматически отвечает сам и переводит диалог обратно в STATE_ACTIVE.

## Деплой на Railway

1. Создать новый проект на Railway
2. Подключить этот репозиторий
3. Добавить плагин **PostgreSQL**
4. Прописать все переменные окружения
5. Railway автоматически запустит `python bot.py`

После деплоя:
- Настроить хук в EnvyCRM → Настройки → Интеграции → Webhook → URL: `https://<railway-url>/envy_hook`
- Убедиться что Wazzup направляет события в EnvyCRM (не напрямую в бота)
